# **Reverse Language**
An esoteric programming language that works in reverse

## Variable definition

```
;<value> = identifier
;45 = MyNumber
```

## Arithmetic

```
;45 23 + = sumResult
;45 23 - = subtractionResult
;45 23 * = multiplicationResult
;45 23 / = divisionResult
```

## Boolean logic

`true` evaluates to `0`  
`false` evaluates to `1`

```
;a b ==
;a b !=
;a b &&
;a b ||
```
